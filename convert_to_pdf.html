📱 PDF转换指南

🎯 如何将markdown文件转换为PDF

方法1：使用在线转换工具（推荐）

  1. 访问 markdown-pdf.com 或 convertio.co
  2. 上传您的 "东京探索之旅.md" 文件
  3. 选择输出格式为PDF
  4. 点击转换，下载生成的PDF文件

方法2：使用Typora软件

  1. 下载安装Typora（支持Windows/Mac/Linux）
  2. 用Typora打开 "东京探索之旅.md" 文件
  3. 点击菜单栏 文件 → 导出 → PDF
  4. 选择保存位置，生成PDF

方法3：使用VS Code

  1. 安装VS Code和"Markdown PDF"插件
  2. 打开markdown文件
  3. 按Ctrl+Shift+P，输入"markdown pdf"
  4. 选择"Markdown PDF: Export (pdf)"

方法4：使用浏览器打印功能

  1. 用支持markdown的编辑器预览文件（如GitHub、GitLab等）
  2. 在浏览器中按Ctrl+P
  3. 选择"打印到PDF"
  4. 调整页面设置，保存PDF

📋 PDF优化建议

  • ✅ 确保图片清晰度适中（避免文件过大）
  • ✅ 设置合适的页边距（建议20mm）
  • ✅ 选择A4纸张尺寸便于打印
  • ✅ 检查中文字体显示是否正常
  • ✅ 测试打印一页确认效果

📸 打印注意事项

  • 建议使用彩色打印以获得最佳视觉效果
  • 图片较多，建议双面打印节省纸张
  • 可以考虑打印成小册子形式便于携带
  • 重要页面可以单独打印并塑封保护

📁 文件信息

  • 主文件：东京探索之旅.md（完整版，约35,000字）
  • 便携版：东京探索之旅_便携版.md（精简版，约7,000字）
  • 建议：完整版用于详细学习，便携版用于现场携带
